## API Reference

This component is built using [Radix UI Separator](https://www.radix-ui.com/primitives/docs/components/separator) primitives. For detailed information, please visit the full API reference.

### Separator

This component is based on the [Separator.Root](https://www.radix-ui.com/primitives/docs/components/separator#root) primitive and includes the following custom props:

| **Prop**                                                                                                       | **Type**                                                       | **Default**    |
| -------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------- | -------------- |
| `orientation` <PopoverInfo>The orientation of the separator.</PopoverInfo>                                     | ~enum~ <PopoverInfo> "horizontal" \| "vertical" </PopoverInfo> | ~"horizontal"~ |
| `decorative` <PopoverInfo>Whether the separator is purely decorative or semantically meaningful.</PopoverInfo> | ~boolean~                                                      | ~true~         |
| `className` <PopoverInfo>Additional class names for styling the separator.</PopoverInfo>                       | ~string~                                                       | <Na/>          |
