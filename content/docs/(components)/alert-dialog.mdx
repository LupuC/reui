## API Reference

This component is built using [Radix UI AlertDialog](https://www.radix-ui.com/primitives/docs/components/alert-dialog) primitives. For detailed information, please visit the full API reference.

### AlertDialog

This component is based on the [AlertDialog.Root](https://www.radix-ui.com/primitives/docs/components/alert-dialog#root) primitive and includes the following custom props:

| **Prop**                                                                                          | **Type**                                                           | **Default** |
| ------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ | ----------- |
| `variant` <PopoverInfo>Determines the visual style of the alert dialog.</PopoverInfo>             | ~enum~ <PopoverInfo> "info" \| "warning" \| "error" </PopoverInfo> | ~info~      |
| `dismissable` <PopoverInfo>Controls whether the alert can be dismissed by the user.</PopoverInfo> | ~boolean~                                                          | ~true~      |
| `className` <PopoverInfo>Additional class names for styling the alert dialog root.</PopoverInfo>  | ~string~                                                           | <Na/>       |

### AlertDialogTrigger
