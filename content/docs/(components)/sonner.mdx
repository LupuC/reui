## API Reference

This component is built using [Sonner](https://sonner.emilkowal.ski/) toast library. For detailed information, please visit the full API reference.

### Toaster

This component extends the Sonner Toaster component with theme integration and custom styling. It accepts all props from the Sonner Toaster component.

| **Prop**                                                                                        | **Type**                        | **Default** |
| ----------------------------------------------------------------------------------------------- | ------------------------------- | ----------- |
| `theme` <PopoverInfo>The theme to use for the toasts. Inherited from next-themes.</PopoverInfo> | ~"light" \| "dark" \| "system"~ | ~"system"~  |
