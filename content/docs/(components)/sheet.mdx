## API Reference

This component is built using [Radix UI Dialog](https://www.radix-ui.com/primitives/docs/components/dialog) primitives. For detailed information, please visit the full API reference.

### Sheet

This component is based on the [Dialog.Root](https://www.radix-ui.com/primitives/docs/components/dialog#root) primitive. For more details, refer to the original documentation.

### SheetTrigger

This component is based on the [Dialog.Trigger](https://www.radix-ui.com/primitives/docs/components/dialog#trigger) primitive. For more details, refer to the original documentation.

### SheetContent

This component is based on the [Dialog.Content](https://www.radix-ui.com/primitives/docs/components/dialog#content) primitive and includes the following custom props:

| **Prop**                                                                               | **Type**                                                                   | **Default** |
| -------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- | ----------- |
| `side` <PopoverInfo>The side from which the sheet appears.</PopoverInfo>               | ~enum~ <PopoverInfo> "top" \| "bottom" \| "left" \| "right" </PopoverInfo> | ~"right"~   |
| `overlay` <PopoverInfo>Whether to show the overlay background.</PopoverInfo>           | ~boolean~                                                                  | ~true~      |
| `close` <PopoverInfo>Whether to show the close button.</PopoverInfo>                   | ~boolean~                                                                  | ~true~      |
| `className` <PopoverInfo>Additional class names for styling the content.</PopoverInfo> | ~string~                                                                   | <Na/>       |
