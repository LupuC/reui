## API Reference

This component is built using [Radix UI Tooltip](https://www.radix-ui.com/primitives/docs/components/tooltip) primitives. For detailed information, please visit the full API reference.

### Tooltip

This component is based on the [Tooltip.Root](https://www.radix-ui.com/primitives/docs/components/tooltip#root) primitive. For full details, visit the official documentation.

### TooltipTrigger

This component is based on the [Tooltip.Trigger](https://www.radix-ui.com/primitives/docs/components/tooltip#trigger) primitive. For full details, visit the official documentation.

### TooltipContent

This component is based on the [Tooltip.Content](https://www.radix-ui.com/primitives/docs/components/tooltip#content) primitive and includes the following custom props:

| **Prop**                                                                               | **Type** | **Default** |
| -------------------------------------------------------------------------------------- | -------- | ----------- |
| `sideOffset` <PopoverInfo>The distance in pixels from the trigger.</PopoverInfo>       | ~number~ | ~4~         |
| `className` <PopoverInfo>Additional class names for styling the content.</PopoverInfo> | ~string~ | ~undefined~ |

### TooltipProvider

This component is based on the [Tooltip.Provider](https://www.radix-ui.com/primitives/docs/components/tooltip#provider) primitive. For full details, visit the official documentation.
