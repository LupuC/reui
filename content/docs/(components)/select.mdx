## API Reference

This component is built using [Radix UI Select](https://www.radix-ui.com/primitives/docs/components/select) primitives. For detailed information, please visit the full API reference.

### Select

This component is based on the [Select.Root](https://www.radix-ui.com/primitives/docs/components/select#root) primitive and includes the following custom props:

| **Prop**                                                                                             | **Type**                                              | **Default** |
| ---------------------------------------------------------------------------------------------------- | ----------------------------------------------------- | ----------- |
| `indicatorPosition` <PopoverInfo>Position of the selection indicator.</PopoverInfo>                  | ~enum~ <PopoverInfo> "left" \| "right" </PopoverInfo> | ~"left"~    |
| `indicatorVisibility` <PopoverInfo>Controls the visibility of the selection indicator.</PopoverInfo> | ~boolean~                                             | ~true~      |
| `indicator` <PopoverInfo>Custom indicator element to replace the default check icon.</PopoverInfo>   | ~ReactNode~                                           | <Na/>       |

### SelectTrigger

This component is based on the [Select.Trigger](https://www.radix-ui.com/primitives/docs/components/select#trigger) primitive and includes the following custom props:

| **Prop**                                                                               | **Type**                                                 | **Default** |
| -------------------------------------------------------------------------------------- | -------------------------------------------------------- | ----------- |
| `size` <PopoverInfo>Controls the size of the select trigger.</PopoverInfo>             | ~enum~ <PopoverInfo> "xs" \| "sm" \| "md" </PopoverInfo> | ~"md"~      |
| `className` <PopoverInfo>Additional class names for styling the trigger.</PopoverInfo> | ~string~                                                 | <Na/>       |

### SelectContent

This component is based on the [Select.Content](https://www.radix-ui.com/primitives/docs/components/select#content) primitive and includes the following custom props:

| **Prop**                                                                               | **Type**   | **Default** |
| -------------------------------------------------------------------------------------- | ---------- | ----------- |
| `position` <PopoverInfo>The positioning strategy to use.</PopoverInfo>                 | ~"popper"~ | ~"popper"~  |
| `className` <PopoverInfo>Additional class names for styling the content.</PopoverInfo> | ~string~   | <Na/>       |

### SelectItem

This component is based on the [Select.Item](https://www.radix-ui.com/primitives/docs/components/select#item) primitive and includes the following custom props:

| **Prop**                                                                            | **Type** | **Default** |
| ----------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the item.</PopoverInfo> | ~string~ | <Na/>       |

The component inherits indicator position and visibility settings from the Select context.

| **Prop**                                                                             | **Type** | **Default** |

### SelectSeparator

| **Prop**                                                                                 | **Type** | **Default** |

| `className` <PopoverInfo>Additional class names for styling the separator.</PopoverInfo> | ~string~ | <Na/>       |
