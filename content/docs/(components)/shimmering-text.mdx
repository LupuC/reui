## API Reference

### ShimmeringText

<PopoverInfo>A text component with beautiful shimmer animation effects that glide across the text.</PopoverInfo>

| Prop           | Type      | Default | Description                                                                                    |
| -------------- | --------- | ------- | ---------------------------------------------------------------------------------------------- |
| `text`         | ~string~  | -       | <PopoverInfo>Text to display with shimmer effect</PopoverInfo>                                 |
| `duration`     | ~number~  | ~2~     | <PopoverInfo>Animation duration in seconds</PopoverInfo>                                       |
| `delay`        | ~number~  | ~0~     | <PopoverInfo>Delay before starting animation</PopoverInfo>                                     |
| `repeat`       | ~boolean~ | ~true~  | <PopoverInfo>Whether to repeat the animation</PopoverInfo>                                     |
| `repeatDelay`  | ~number~  | ~0.5~   | <PopoverInfo>Pause duration between repeats in seconds</PopoverInfo>                           |
| `className`    | ~string~  | -       | <PopoverInfo>Custom CSS classes to apply to the component</PopoverInfo>                        |
| `startOnView`  | ~boolean~ | ~true~  | <PopoverInfo>Whether to start animation when component enters viewport</PopoverInfo>           |
| `once`         | ~boolean~ | ~false~ | <PopoverInfo>Whether to animate only once</PopoverInfo>                                        |
| `inViewMargin` | ~string~  | -       | <PopoverInfo>Margin for in-view detection (rootMargin)</PopoverInfo>                           |
| `spread`       | ~number~  | ~2~     | <PopoverInfo>Shimmer spread multiplier - affects the width of the shimmer effect</PopoverInfo> |
| `color`        | ~string~  | -       | <PopoverInfo>Base text color (overrides default theme colors)</PopoverInfo>                    |
| `shimmerColor` | ~string~  | -       | <PopoverInfo>Shimmer gradient color (overrides default theme colors)</PopoverInfo>             |
