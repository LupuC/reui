## API Reference

This component is built using [Radix UI Dialog](https://www.radix-ui.com/primitives/docs/components/dialog) primitives. For detailed information, please visit the full API reference.

### Dialog

This component is based on the [Dialog.Root](https://www.radix-ui.com/primitives/docs/components/dialog#root) primitive. For more details, refer to the original documentation.

### DialogTrigger

This component is based on the [Dialog.Trigger](https://www.radix-ui.com/primitives/docs/components/dialog#trigger) primitive. For more details, refer to the original documentation.

### DialogPortal

This component is based on the [Dialog.Portal](https://www.radix-ui.com/primitives/docs/components/dialog#portal) primitive. For more details, refer to the original documentation.

### DialogOverlay

This component is based on the [Dialog.Overlay](https://www.radix-ui.com/primitives/docs/components/dialog#overlay) primitive and includes the following custom props:

| **Prop**                                                                               | **Type** | **Default** |
| -------------------------------------------------------------------------------------- | -------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the overlay.</PopoverInfo> | ~string~ | <Na/>       |

### DialogContent

This component is based on the [Dialog.Content](https://www.radix-ui.com/primitives/docs/components/dialog#content) primitive and includes the following custom props:

| **Prop**                                                                                      | **Type**  | **Default** |
| --------------------------------------------------------------------------------------------- | --------- | ----------- |
| `close` <PopoverInfo>Controls whether the close button is displayed.</PopoverInfo>            | ~boolean~ | ~true~      |
| `className` <PopoverInfo>Additional class names for styling the dialog content.</PopoverInfo> | ~string~  | <Na/>       |

### DialogClose

This component is based on the [Dialog.Close](https://www.radix-ui.com/primitives/docs/components/dialog#close) primitive. For more details, refer to the original documentation.
