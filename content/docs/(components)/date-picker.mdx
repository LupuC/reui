## API Reference

This component is built using [react-day-picker](https://react-day-picker.js.org/) and [Radix UI Popover](https://www.radix-ui.com/primitives/docs/components/popover) primitives. For detailed information about the calendar functionality, please visit the react-day-picker documentation.

### Calendar

This component is based on the [DayPicker](https://react-day-picker.js.org/basics/customization) component from react-day-picker. For all available props, refer to the react-day-picker documentation. The component includes the following custom styling props:

| **Prop**                                                                                          | **Type** | **Default**        |
| ------------------------------------------------------------------------------------------------- | -------- | ------------------ |
| `className` <PopoverInfo>Additional class names for styling the calendar container.</PopoverInfo> | ~string~ | ~undefined~        |
| `classNames` <PopoverInfo>Custom class names for individual calendar elements.</PopoverInfo>      | ~object~ | ~See calendar.tsx~ |

### DatePickerTrigger

The trigger button that opens the date picker popover.

| **Prop**                                                                                      | **Type**  | **Default**     |
| --------------------------------------------------------------------------------------------- | --------- | --------------- |
| `asInput` <PopoverInfo>Style the trigger as an input field.</PopoverInfo>                     | ~boolean~ | ~false~         |
| `placeholder` <PopoverInfo>Placeholder text when no date is selected.</PopoverInfo>           | ~string~  | ~"Pick a date"~ |
| `className` <PopoverInfo>Additional class names for styling the trigger button.</PopoverInfo> | ~string~  | ~undefined~     |

### DatePickerContent

The popover content that contains the calendar. This component extends Radix UI's Popover.Content.

| **Prop**                                                                                       | **Type**                                                         | **Default** |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- | ----------- |
| `className` <PopoverInfo>Additional class names for styling the popover content.</PopoverInfo> | ~string~                                                         | ~undefined~ |
| `align` <PopoverInfo>Alignment of the popover relative to the trigger.</PopoverInfo>           | ~enum~ <PopoverInfo> "start" \| "center" \| "end" </PopoverInfo> | ~"center"~  |

### DatePickerPresets

A component for displaying preset date ranges.

| **Prop**                                                                                         | **Type**                                        | **Default** |
| ------------------------------------------------------------------------------------------------ | ----------------------------------------------- | ----------- |
| `presets` <PopoverInfo>Array of preset date ranges.</PopoverInfo>                                | ~Array\<\{ label: string; range: DateRange \}>~ | <Na/>       |
| `onSelect` <PopoverInfo>Callback function when a preset is selected.</PopoverInfo>               | ~(range: DateRange) => void~                    | <Na/>       |
| `className` <PopoverInfo>Additional class names for styling the presets container.</PopoverInfo> | ~string~                                        | ~undefined~ |
