## API Reference

This component is built using [Radix UI Popover](https://www.radix-ui.com/primitives/docs/components/popover) primitives. For detailed information, please visit the full API reference.

### Popover

This component is based on the [Popover.Root](https://www.radix-ui.com/primitives/docs/components/popover#root) primitive. For more details, refer to the original documentation.

### PopoverTrigger

This component is based on the [Popover.Trigger](https://www.radix-ui.com/primitives/docs/components/popover#trigger) primitive. For more details, refer to the original documentation.

### PopoverContent

This component is based on the [Popover.Content](https://www.radix-ui.com/primitives/docs/components/popover#content) primitive and includes the following custom props:

| **Prop**                                                                                       | **Type**                                                         | **Default** |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------- | ----------- |
| `align` <PopoverInfo>The preferred alignment against the trigger.</PopoverInfo>                | ~enum~ <PopoverInfo> "start" \| "center" \| "end" </PopoverInfo> | ~"center"~  |
| `sideOffset` <PopoverInfo>The distance in pixels from the trigger.</PopoverInfo>               | ~number~                                                         | ~4~         |
| `className` <PopoverInfo>Additional class names for styling the popover content.</PopoverInfo> | ~string~                                                         | <Na/>       |
